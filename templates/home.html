Temperature Data API

Get the mean temperature value of India from 1901 to 2017

url format : http://127.0.0.1:5000/api/in/_year_/_month_

Example url: http://127.0.0.1:5000/api/in/2017/jan



Get the mean temperature value at different European weather stations from 1936 to 2021

url format : http://127.0.0.1:5000/api/eu/_stationcode_/_YYYYMMDD_

Example url: http://127.0.0.1:5000/api/eu/100/20010331
